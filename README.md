# Used Car Market Analysis

## Overview
This project uses web scraping, data cleaning and exploratory data analysis to study the used car market for Toyota, Ford and BMW vehicles.

The project investigates:
- pricing trends
- model availability
- fuel type and transmission patterns
- the relationship between mileage and price

## Dataset
The final cleaned dataset contains:
- 737 observations
- 7 variables
- 3 brands
- 70 distinct car models

Brands analysed:
- Toyota
- Ford
- BMW

## Objectives
- Scrape used car listings from the web
- Clean and standardise the raw data
- Handle missing values and outliers
- Compare prices across brands and models
- Analyse the effect of fuel type and transmission
- Examine the relationship between mileage and price

## Key Findings
- BMW vehicles had the highest average price (£48,201)
- Ford vehicles had an average price of £10,371
- Toyota vehicles had an average price of £9,037
- Mileage and price showed a moderate negative correlation of approximately -0.65
